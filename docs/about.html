Himawari Downloader

Easy, free and open source!

Himawari Downloader originated when i wanted to create a timelapse using the images from Himawari. I could not find a way to download the images in all their glory (11000px). Since I know a lot about Java I chose it for this project although it might not be optimal. The project is a just for fun project and I have no intention making it a big deal. Neither do I know if you are allowed to download the images this way. Use at your own risk!

The program might contain some minor or major bugs. If you can find one please report it here: Issues
The GitHub repository can be found here: Himawari Downloader

Have fun!
